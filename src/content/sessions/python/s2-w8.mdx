Sprint 2 starts with the internet. You will make your first HTTP request in Python — asking a real server for real data — and parse the JSON it sends back. By the end of the session, one function call prints the current weather in Medellín. The skills you built in Sprint 1 (functions, loops, dicts) are exactly what you use here.

**Before the session:** get a free API key from openweathermap.org and install `requests` with `pip install requests`.
