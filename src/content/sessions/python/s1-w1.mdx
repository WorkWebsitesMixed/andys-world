Before anyone touches an installer, the teacher runs the Week 2 hook script on the projector — seven lines of Python that print your name with a random job title. That's the goal. By the end of this session your machine is ready, and you have already run your first program with your name inside it.

Setup takes the whole hour. If you don't finish, no problem — book five minutes before next session and we'll sort it out. The only requirement for Week 2 is a working `python W1_first_script.py` run.
