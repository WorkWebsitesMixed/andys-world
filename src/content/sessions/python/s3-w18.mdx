APIs don't have to be weather and news. This session you build a movie explorer: search for any film, get ratings and plot details from OMDb, and save your favourites to a local JSON file that persists between runs. The fun part is picking your own movies to test it with.

**Before the session:** get a free API key from omdbapi.com. Add it to `.env` as `OMDB_API_KEY`.
