This is the session where you talk to an AI. Not through a chat interface — through code you write. You will send a question to Gemini from your terminal and get a real answer back. You will also learn the single most important habit in API development: never put secret keys in your code.

**Before the session:** get a free API key from aistudio.google.com and run `pip install google-generativeai python-dotenv`.
