Sprint 3 is about automation — making the computer do boring things so you don't have to. The first boring thing: sorting files. You will write a script that looks at every file in a folder, decides what category it belongs to (images, documents, code, etc.), and moves it into the right subfolder. Run it once; a messy Downloads folder becomes organised in milliseconds.
