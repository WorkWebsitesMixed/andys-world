One API call gives you weather. Another gives you news. Combine them, and you have a morning digest that runs the moment you open your terminal. This session you compose two separate API functions into one useful tool — and add a little ASCII art to make it look good.

**Before the session:** get free keys from openweathermap.org and newsapi.org. Add them to your `.env` file as `WEATHER_API_KEY` and `NEWS_API_KEY`.
