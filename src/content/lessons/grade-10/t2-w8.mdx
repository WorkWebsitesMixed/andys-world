<KeyTerms terms={[
  { term: "Pilot hole", definition: "A small-diameter hole drilled before a screw, reducing the risk of the timber splitting. The pilot hole diameter is slightly smaller than the screw shank, so the thread still bites. Always drill a pilot hole when screwing into end grain or near the edge of a board." },
  { term: "End grain", definition: "The face of timber that shows the cross-section of the wood fibres (the cut end of the plank). Screws hold poorly in end grain — the fibres run parallel to the screw, so there is nothing to bite into. Solution: use a corner block, add a pilot hole, or choose a different joint design." },
  { term: "Checking for square", definition: "After assembly, place a steel rule or tape diagonally across the assembled frame from corner to corner. Measure both diagonals. If they are equal, the frame is square. If one diagonal is longer, the frame is parallelogram-shaped and must be corrected before the adhesive cures." },
  { term: "Contact adhesive — bond timing", definition: "Apply contact adhesive to both surfaces. Allow to dry until tacky (not wet) — approximately 5–10 minutes. Then press together only once. The bond forms immediately on first contact and cannot be repositioned. Pressing wet surfaces together produces a weak bond." },
]} />

<Section title="Joining sequence for the desk organizer" icon="🔧" id="joining">

### Pine frame — butt joint with PVA + screws

1. Pre-drill clearance holes in the top component and pilot holes in the lower component.
2. Apply PVA to both wood faces.
3. Clamp lightly, then drive screws.
4. Check square using diagonal measurement.
5. Wipe excess PVA immediately with a damp cloth.
6. Leave clamped for 30 minutes minimum.

### Acrylic panel to pine frame — contact adhesive

1. Ensure both surfaces are clean and dust-free.
2. Apply contact adhesive to both surfaces (pine face + acrylic face).
3. Wait 5–10 minutes until surfaces are tacky (dry to touch but not cured).
4. **Ventilate:** contact adhesive contains solvent — keep workshop door open.
5. Press together only once, firmly and evenly.
6. Allow 60 minutes before applying any load.

### Adding internal dividers — wood screws only

Dividers are often added without adhesive so they can be repositioned later. Pre-drill and countersink; drive screws from the outside of the frame into the end grain of the divider. Add a corner block inside for rigidity.

</Section>

<WorkedExample title="Paper 3 Section A — joining questions worked">

**Q:** "Give one reason why PVA is not suitable for joining an acrylic panel to a pine frame. [1 mark]"

**Strong answer:** "PVA does not form an adhesive bond with polymer surfaces — it bonds wood fibres to wood fibres only. On acrylic, the PVA cures without adhering, so the joint fails."

**Q:** "Name the type of adhesive suitable for bonding acrylic to pine, and give one safety precaution. [2 marks]"

**Strong answer:** "Contact adhesive. [1 mark] The workshop must be ventilated because contact adhesive releases solvent fumes that are harmful if inhaled. [1 mark]"

**Pattern:** adhesive name or join type → named reason or specific precaution.
</WorkedExample>

3. Join all pine frame components (butt joints: PVA + screws, pilot holes first).
4. Check square after every corner joint.
5. Attach the acrylic panel using contact adhesive (both surfaces coated, wait until tacky, press once).
6. Add internal dividers if time allows (wood screws only, corner block inside).
7. Mark any components not yet joined — carry forward to W9.

**Squareness check + clamp (70–75 min):**
8. Measure both diagonals of the assembled frame. Record: "Diagonal A = ___ mm; Diagonal B = ___ mm; Difference = ___ mm."
9. Clamp the frame if still wet.

<Support title="If the frame is not square">

While the PVA is still wet, push the longer diagonal corner toward the shorter corner — this effectively pulls the frame into a square. Clamp diagonally across the longer diagonal to hold the correction while the glue cures. Check with both diagonals again before leaving the workshop.
</Support>
