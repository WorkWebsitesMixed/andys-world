Electronics starts from zero this week. You do not need any prior knowledge — every symbol and concept is introduced before you draw or build anything. By the end of the session you will have a circuit symbol reference sheet you made yourself and an understanding of how the Firefighter Barbie circuit's three main stages connect as a system.

<Callout type="exam" title="IPO and specific language" id="why">
  Paper 1 Section A regularly asks you to label a circuit using the input-process-output model and to name the function of each component. Generic answers — "it makes the circuit work" — earn zero. You must name the component, state what it does, and use the correct technical term: thermistor, transistor, motor — never "sensor thing" or "switch part."
</Callout>

<Section title="Circuit symbols and conventions" icon="🔧" id="symbols">

### The eight symbols you must know

| Component | Symbol description | Function |
|---|---|---|
| Cell (single) | Long thin line (positive) + short thick line (negative) | Single power source |
| Battery | Multiple cell symbols in series | Multiple cells providing higher voltage |
| Switch (open) | Line with a gap and a lever | Breaks the circuit when open |
| Resistor | Rectangle | Limits current flow |
| LED | Diode triangle with two arrows pointing outward | Emits light when current flows in forward direction |
| Motor | Circle with M inside | Converts electrical energy to rotational mechanical energy |
| Thermistor | Resistor rectangle with a diagonal arrow | Resistance changes with temperature |
| NPN transistor | Circle with base, collector, emitter lines | Amplifies or switches current — controlled by base current |

Draw each symbol on your reference sheet. Size them consistently. A symbol that is too small or distorted will not be credited in an exam.

### Conventional current and conductors

Current flows from positive to negative terminal in a circuit diagram (conventional direction). The wire (conductor) allows current to flow; the plastic insulation around it prevents current leaking to unwanted paths.

In the Firefighter Barbie circuit: the battery provides the EMF; current flows through the circuit to the motor or LED.

</Section>

<WorkedExample title="Labelling a circuit with IPO">
**Circuit:** Cell → switch → resistor → LED → back to cell.

**IPO labelling:**
- **Input:** Switch (the human trigger — on/off control)
- **Process:** Resistor (limits current to protect the LED)
- **Output:** LED (produces light — the visible result)

**Applied to Firefighter Barbie:**
- **Input:** Thermistor (detects heat — changes resistance as temperature rises)
- **Process:** Transistor circuit (makes the switching decision — turns on when base voltage exceeds threshold)
- **Output:** DC motor (converts electrical energy to rotation — drives the crank or cam mechanism)

The IPO model helps you see that each component has a role in the system — not just a name.
</WorkedExample>

<Section title="Series vs parallel circuits" icon="🔧" id="series-parallel">

### Series circuit

All components in one single loop. The same current flows through every component.

**Consequence:** If you add more components, each one shares the total supply voltage — each component gets less voltage. Two LEDs in series: each gets half the supply voltage, so they are dimmer than a single LED.

**If one breaks:** The circuit is open and nothing works.

### Parallel circuit

Components in separate branches. The supply voltage appears across every branch.

**Consequence:** Each branch gets the full supply voltage regardless of how many branches there are. Two LEDs in parallel: each gets full voltage, so they are as bright as a single LED. Total current from the battery doubles.

**If one branch breaks:** Other branches continue to work.

### Exam practice — Section A quick-fire

The bonus session is a Section A quick-fire covering both mechanisms (Weeks 4–5) and electronics (this week). For each question: name the specific component or motion type, give a reason using technical language, and never leave an answer blank.

**Rewriting exercise:** "A crank is used to change the motor's movement" → rewrite to name the exact input motion type, the output motion type, and the connecting mechanism component that makes the conversion.

</Section>

**W session:**
1. Complete your own circuit symbol reference sheet: draw all eight symbols neatly at a consistent size. Label each with name and function.
2. Draw a switch → resistor → LED circuit with IPO labels.
3. Draw a series circuit with two LEDs and a cell. Then draw a parallel circuit with the same two LEDs. Label which LEDs will be brighter and why.

**B session:**
4. Six Section A quick-fire questions: three on mechanisms (motion types, crank, cam) and three on electronics (symbols, IPO, series/parallel). Self-mark.
5. Rewrite the two answers with the lowest marks, removing any generic words. Show what you changed.
</Activity>
