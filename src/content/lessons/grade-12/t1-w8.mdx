  The Chemistry lab report currently written in Word has three persistent quality problems: chemical formulae written as plain text (`H2O` instead of H₂O), figure numbers that go out of sync when figures are reordered, and a manually formatted reference list. <Section title="LaTeX document structure" icon="💻" id="lesson">

### The skeleton every document starts with

```latex
\documentclass[12pt]{article}
\usepackage{geometry}
\usepackage[utf8]{inputenc}

\begin{document}

Your content goes here.

\end{document}
```

Everything before `\begin{document}` is the **preamble**. Everything between `\begin{document}` and `\end{document}` is the **body**.

### Key commands

| Command | What it does |
|---|---|
| `\section{Title}` | Creates a numbered section heading |
| `\subsection{Title}` | Creates a numbered sub-heading |
| `\textbf{text}` | **Bold** text |
| `\textit{text}` | *Italic* text |
| `\emph{text}` | Emphasis — usually italic, context-aware |
| Blank line | Starts a new paragraph |
| `%` | Comment — LaTeX ignores everything after this on the line |

### Packages: adding extra features

Packages extend what LaTeX can do. Load them in the preamble with `\usepackage{name}`.

```latex
\usepackage{geometry}          % controls page margins
\usepackage[utf8]{inputenc}    % handles accented characters (á, é, etc.)
```

### Reading a compile error

When the PDF doesn't build, Overleaf shows a red error panel. Click it to see the log. The key information is:

- **Line number** — go to that line in the editor.
- **Error message** — common ones: "Missing } inserted" (unclosed brace), "Undefined control sequence" (typo in command name).

**Rule:** every `{` needs a matching `}`. Every `\begin{env}` needs a matching `\end{env}`.

### Common first-session mistakes

| Mistake | What to check |
|---|---|
| `/section` instead of `\section` | Backslash `\`, not forward slash `/` |
| Edited PDF pane | Source pane is on the left — that's where you type |
| Changed something but PDF didn't update | Did you click **Recompile**? |
| Wrong quotation marks | Use ` `` ` for opening, `''` for closing — not `"text"` |

<Support title="LaTeX first-session survival guide">
If the PDF won't compile:

1. Click the error panel in Overleaf.
2. Find the line number in the error message.
3. Go to that line in the source.
4. Look for: unclosed `{`, missing `}`, typo in a command name (`\scetion` instead of `\section`).
5. Fix it. Compile again.

Don't delete sections and restart — that makes it harder to find the error.
</Support>
