Four packages, four new powers. This week you add everything that makes a Chemistry lab report look like a real scientific document: proper chemical equations, figures that number themselves, publication-style tables, and a bibliography that formats automatically. Introduce them one at a time.

<Callout type="tip" title="Why packages?" id="why">
  Each package solves a specific problem that plain LaTeX can't handle. `mhchem` makes chemical formulae correct automatically. `graphicx` handles images. `booktabs` produces clean, journal-style tables. `biblatex` manages references so the bibliography is never manually formatted again.
</Callout>

<KeyTerms terms={[
  { term: "\\usepackage{name}", definition: "The command to load a package in the preamble. Load all packages before \\begin{document}." },
  { term: "mhchem", definition: "A package for typesetting chemical formulae and equations. The key command is \\ce{} — everything inside is treated as chemistry." },
  { term: "graphicx", definition: "A package for inserting images. Used with the figure environment for captioned, numbered figures." },

  { term: "biblatex", definition: "A package for managing references. Sources go in a separate .bib file; citations appear inline with \\cite{}; the full list prints with \\printbibliography." },

<Section title="Package 1: mhchem — chemical equations" icon="💻" id="mhchem">

Add to the preamble:
```latex
\usepackage[version=4]{mhchem}
```

The core command is `\ce{}`. Everything inside is treated as chemistry — subscripts, superscripts, reaction arrows and ionic charges are all handled automatically.

| What you type | What appears |
|---|---|
| `\ce{H2O}` | H₂O (subscript 2) |
| `\ce{KClO3}` | KClO₃ |
| `\ce{2KClO3 -> 2KCl + 3O2}` | Balanced equation with arrow |
| `\ce{Ag+}` | Ag⁺ (superscript +) |
| `\ce{CO3^2-}` | CO₃²⁻ |

**Task:** open your lab report. Find the five chemical equations in the Discussion section currently written as plain text. Replace each with `\ce{}`. Compile and compare the output.

</Section>

<Section title="Package 2: graphicx — figures" icon="💻" id="graphicx">

Add to the preamble:
```latex
\usepackage{graphicx}
\usepackage{float}
```

The figure environment:
```latex
\begin{figure}[H]
  \centering
  \includegraphics[width=0.6\textwidth]{filename.jpg}
  \caption{Potassium chlorate melting in the test tube, before decomposition.}
  \label{fig:decomp1}
\end{figure}
```

- `[H]` (from the `float` package) forces the figure to appear exactly where you placed it, not wherever LaTeX decides to float it.
- `width=0.6\textwidth` sets the image to 60% of the text width. Adjust this number to resize.
- `\caption{}` adds the numbered caption below the figure — the number is automatic.
- `\label{fig:decomp1}` gives the figure a name you can reference from the text.

**Cross-reference in the text:**
```latex
As shown in Figure~\ref{fig:decomp1}, the potassium chlorate begins to melt before decomposition begins.
```

The `~` prevents a line break between "Figure" and the number. The number updates automatically if figures are reordered.

**Task:** upload two lab photos to your Overleaf project. Insert both as figures with captions and labels. Cross-reference each one from the Results section.

</Section>

<Section title="Package 3: booktabs — tables" icon="💻" id="booktabs">

Add to the preamble:
```latex
\usepackage{booktabs}
```

A basic results table:
```latex
\begin{table}[H]
  \centering
  \caption{Mass of oxygen collected at each time interval.}
  \label{tab:results}
  \begin{tabular}{lcc}
    \toprule
    Time (min) & Mass before (g) & Mass after (g) \\
    \midrule
    2  & 10.42 & 10.38 \\
    4  & 10.38 & 10.31 \\
    6  & 10.31 & 10.20 \\
    \bottomrule
  \end{tabular}
\end{table}
```

`\toprule`, `\midrule` and `\bottomrule` produce the three rules of a publication-style table. No vertical lines — this is intentional and correct for scientific tables.

In the column definition `{lcc}`: `l` = left-aligned, `c` = centred. Use `r` for right-aligned numbers.

</Section>

<Section title="Package 4: biblatex — references" icon="💻" id="biblatex">

Add to the preamble:
```latex
\usepackage[style=numeric,sorting=none]{biblatex}
\addbibresource{refs.bib}
```

Create a new file in Overleaf called `refs.bib`. One entry per reference:
```
@article{brown2020,
  author = {Brown, James},
  title  = {Thermal decomposition reactions in secondary education},
  year   = {2020},
  url    = {https://example-journal.org/article/...}
}
```

**Cite inline** (in your body text):
```latex
This reaction is well documented in the literature \cite{brown2020}.
```

**Print the bibliography** (at the end of your document, before `\end{document}`):
```latex
\printbibliography
```

The bibliography is generated automatically from the entries you cite. You never format it manually.

**Task:** enter three references from your lab report into `refs.bib`. Cite them in the Introduction. Compile and check the bibliography appears.

</Section>

<Support title="Common errors this session">
**mhchem error — "Package mhchem not found":** check the spelling in `\usepackage[version=4]{mhchem}`. The option `[version=4]` is required.

**Figure doesn't appear where expected:** add the `float` package and use `[H]` as the position argument.

**biblatex compilation takes two passes:** if the bibliography doesn't appear, click Compile twice (or switch to XeLaTeX in Overleaf settings if errors persist).

**Table column count mismatch:** count the `&` separators — you need exactly one fewer `&` than you have columns. `\toprule`, `\midrule`, `\bottomrule` end with `\\` only after the last `&` on that row.
</Support>
