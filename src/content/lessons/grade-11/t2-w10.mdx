<Section title="Paper 4 exam strategy" icon="🎯" id="strategy">

### Before starting

1. Write your name and candidate number on every answer sheet immediately.
2. Read the Section B context (the design scenario) before reading Section A questions. The context primes your thinking.
3. Note the total marks available for each section. Allocate time before the clock starts.

### Section A approach (40 minutes)

- **1-mark questions:** One specific correct statement. One sentence.
- **2-mark questions:** Two distinct points, or one explained point with specific detail.
- **3–4 mark calculations:** Write formula, convert units, substitute, calculate, state unit.
- **Triage:** Mark any question you skip. Return to it in the last 5 minutes if time allows.

### Section B approach (80 minutes)

- **Read each sub-question before answering any.** Sub-parts (a)(i), (a)(ii), (b)(i) etc. often build on each other. Reading ahead prevents contradicting yourself.
- **Structure 6+ mark answers.** For a 6-mark question: plan 6 distinct points in the margin before writing. Minimum one sentence per mark.
- **Link to context.** Section B marks are almost always awarded for applying knowledge to the specific design context given — not for generic statements.

### After finishing

- **Review blank answers.** A blank earns zero. A partial attempt may earn 1 mark.
- **Check units on all calculations.**
- **Check you have answered all sub-parts.** Multi-part questions are easy to answer partially without noticing.

</Section>

<Section title="Content summary: everything the mock will test" icon="📚" id="content">

### T1 topics (W1–W6)

- Design brief, user profile, function points.
- Structures: frame/shell/mass; triangulation; five forces; FoS calculation; beam reactions.
- Materials: hardwoods, softwoods, manufactured boards; ferrous/non-ferrous metals; composites; 6Rs.
- Electronics: IPO model; circuit symbols; switch types; current-limiting resistor calculation.

### T2 topics (W1–W9)

- Development moves; iterative design; before-and-after annotation.
- First-angle orthographic projection; line types; dimensioning rules.
- Materials testing: bending stiffness, hardness, machinability; selection matrix.
- Manufacture: marking out, datum edge, kerf, dry fit, squareness check.
- Surface finishing: MDF sealing, paint sequence.
- Testing and evaluation: specification testing, met/partially met judgements.
- Electronics 2: NTC thermistor, LDR, voltage divider, transistor switch.
- Sustainability: 6Rs (all six with product-specific examples).

</Section>

<Activity title="Full Paper 4 mock" time="120 min" grouping="Individual (exam conditions)" materials={["Paper 4 mock paper (teacher-provided)", "Black pen", "Calculator", "Ruler"]}>

**Before starting (5 min):** Write candidate information. Read Section B context. Allocate time: Section A = 40 minutes, Section B = 75 minutes, review = 5 minutes.

**Section A (40 min):** Complete all short-answer questions. Show all working for calculations. Triage any question you cannot answer — mark it and return at the end.

**Section B (75 min):** Read each sub-question before answering the first. Structure extended answers with clear points. Link all answers to the Section B context.

**Review (5 min):** Return to any blanks from the triage list. Check units on all calculations. Check all sub-parts have been answered.
</Activity>

<Section title="Post-mock analysis (after papers are returned)" icon="📊" id="analysis">

When your marked paper is returned, complete this analysis:

1. **Score by topic:** How many marks were earned in each topic area? Record: structures / materials / electronics / sustainability / design process.

2. **Loss category:** For each missed mark, categorise: (a) knowledge gap — I did not know this; (b) vocabulary — I used a generic term instead of a specific property; (c) calculation error — method correct but arithmetic wrong; (d) time — I did not attempt this.

3. **T3 priority list:** Rank topic areas by marks lost. The topic with the most losses becomes priority 1 for T3 revision. Write your top three topics before leaving the lesson.

4. **Action plan:** For each priority topic, identify: which textbook section covers it; which past paper questions test it; when you will revise it in T3.
</Section>
