Your folder tells the examiner everything they know about your project. They cannot see you work, they cannot ask you questions, and they cannot assume competence — they can only see what is on the pages. This bonus session is for systematically checking the C5 section (working drawings and plans) against the mark scheme and adding anything that is missing.

<Callout type="exam" title="Why this matters" id="why">
  <ExamLink to="project">Component 2</ExamLink> criterion **C5 (working drawings and plans, 15 marks)** is assessed entirely on the folder. Unlike C6, where the physical product adds context, C5 has no physical product to look at — only the drawings, cutting list, and plan. A gap in C5 evidence is a gap in marks.
</Callout>

<KeyTerms terms={[
  { term: "Mark scheme bandwidth", definition: "C5 marks are awarded in bands (e.g. 0–5, 6–10, 11–15). To reach the top band, all three elements must be present: working drawings (with orthographic views and full dimensions), a cutting list, and a production plan. Missing one element drops the award to the middle or lower band." },
  { term: "Annotation vs dimension", definition: "Annotations explain what a feature is or what it does. Dimensions state a measurement in millimetres. Both are required in working drawings — annotations explain the design; dimensions enable manufacture. A beautiful drawing with no dimensions cannot be used to make anything." },
  { term: "C5 audit", definition: "A systematic check of the C5 folder section against the mark scheme criteria, identifying missing or incomplete evidence and adding it before submission. An audit is more systematic than a quick read-through — it uses a checklist." },
]} />

<Section title="C5 audit checklist" icon="📋" id="audit">

### Working drawings

For each drawing in your set, check:
- [ ] Title block: product name, student name, date, scale, material
- [ ] Third-angle projection symbol present
- [ ] Minimum three views: front, side, top (plan)
- [ ] Overall dimensions on all three views
- [ ] Detail dimensions: joints, holes, peg positions, circuit panel
- [ ] Hidden detail lines (dashed) for internal features
- [ ] Tolerances where relevant (slot widths, hole diameters)
- [ ] Scale consistent across all views and noted in title block

### Cutting list

- [ ] All components listed (every part that needs to be cut or made)
- [ ] Material stated for each component
- [ ] Quantity for each component
- [ ] All three dimensions (L × W × T) for each component
- [ ] Dimensions match the working drawings

### Production plan

- [ ] All major manufacturing steps listed in sequence
- [ ] Time estimate for each step
- [ ] Session allocation (which T3 session each step belongs to)
- [ ] Dependencies noted (e.g. "assembly cannot begin before all panels are cut")

</Section>

<WorkedExample title="C5 audit result — cyclist gear organiser">

**Working drawings audit:**
- Title block: complete on all drawings.
- Third-angle symbol: present on main assembly drawing; MISSING on detail drawings. Fix: add symbol to two detail drawings.
- Views: all three present on main drawing. Side view missing on bracket drawing. Fix: add side view.
- Dimensions: overall dimensions on all views. Slot width tolerance missing (noted as 5.8 mm but no ±tolerance). Fix: add ±0.1 mm tolerance note.

**Cutting list:** 6 components listed. Dimensions match drawings. Switch bracket dimensions need updating (actual print was 59 × 39, not 60 × 40 as listed). Fix: update cutting list.

**Production plan:** All steps listed. Time estimates present. Session allocation column missing. Fix: add session column.

**Total gaps found:** 5 items. Estimated time to fix: 35 minutes. All fixable.
</WorkedExample>

<Activity title="C5 audit + gap filling" time="45 min" grouping="Individual" materials={["C5 folder section", "C5 audit checklist (above)", "Pencil, technical pen, ruler"]}>

**Audit (0–15 min):**
1. Work through the C5 audit checklist. Mark each item: present (P), partial (P/), or missing (M).
2. Count: how many missing items? Can they all be fixed in the remaining 30 minutes?

**Gap filling (15–40 min):**
3. Fix each missing item in priority order (marks-critical first):
   - Third-angle symbols on any drawing missing them
   - Missing views (add a view if a drawing only has two)
   - Missing dimensions (check the working drawing measurement against the product)
   - Missing cutting list dimensions or mismatched values
   - Missing production plan session column

**Final review (40–45 min):**
4. Read through the complete C5 section as if seeing it for the first time. Does it make sense? Can a manufacturer use these drawings and list to make your product without asking you any questions?
</Activity>

<Support title="Can't add a missing view in 10 minutes?">

**Working drawing missing the top view?** You can add it to the existing sheet in the available white space (with a clear label), or add a supplementary A4 sheet with the missing view only. It does not need to match the format of the original drawing exactly — it needs to be dimensioned and linked to the front view with a label.

**Cutting list dimension missing because you forgot to measure?** Measure the actual part now with calipers and use the actual measured dimension. Note if this differs from the design dimension. Update both the cutting list and the drawing if there is a discrepancy.
</Support>
