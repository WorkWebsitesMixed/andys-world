The making phase begins. Today you move from paper and screen to material and machine. The laser cutter is booked, the material is measured, the file is ready. Your job is to set up, cut, and document. The making log starts today — every session from here needs a photograph and a brief note.

<Callout type="exam" title="Why this matters" id="why">
  <ExamLink to="project">Component 2</ExamLink> criterion **C6 (making, 30 marks)** is the largest single criterion in your IGCSE project. Marks come from: quality of manufacture, accuracy to the working drawing, evidence of safe working, and a photographic making log. Today starts the log. A log with no Week 2 entry is already missing evidence.
</Callout>

<Section title="First session in the makerspace" icon="🔨" id="making">

### Setting up safely

Before you touch any machine:
1. **PPE:** safety glasses on, tied-back hair, no loose clothing.
2. **Emergency stop:** identify the location before switching the machine on.
3. **Material check:** measure the actual material thickness with calipers. If it differs from your file by more than 0.5 mm, adjust slot dimensions before cutting.
4. **File check:** open the laser file. Verify cut layer colour, no overlapping paths, parts nested inside the sheet boundary.

### Laser cutting sequence

1. Run a test cut in a corner of the scrap area: one small square of the same material. Check it cuts cleanly through without charring excessively.
2. If settings are correct, load your real file and start the cut job.
3. Do not leave the machine unattended during cutting.
4. After cutting: wait for the extraction fan to clear the chamber before opening the lid.

### Starting the making log

After the laser cut: photograph the cut parts on the machine bed before removing them. Then write:
- Session: T3 W2
- Parts cut: [list by name]
- Process: laser cut, settings used
- Quality check: parts measured — do they match the cutting list dimensions?
- Problems: anything that did not go as planned

</Section>

<Activity title="Laser cut + making log entry" time="90 min" grouping="Individual (safety briefing first; teacher and technician present)" materials={["Laser cutter (booked)", "Material sheet (measured)", "PPE", "Calipers", "Camera (phone)", "Making log sheet or folder page"]}>

**Safety briefing (0–10 min):**
1. Full pre-use briefing from teacher: machine operation, ventilation, fire protocol, emergency stop.

**Setup (10–20 min):**
2. Measure material with calipers. Check against file slot dimensions.
3. Open laser file. Verify cut layer, paths, nesting.

**Test cut (20–30 min):**
4. Run a test cut in the scrap area. Check: cuts cleanly through, no excessive charring, cut width correct.
5. If needed, adjust settings and re-test.

**Main cut job (30–70 min):**
6. Load real file. Start cut job. Monitor throughout.
7. After cutting: wait for extraction to clear, open lid, photograph parts on the bed.

**Quality check + making log (70–90 min):**
8. Measure two critical dimensions on each cut part. Record results.
9. Write making log entry: session, completed, quality check, problems.
10. Start 3D print job if machine available and print is in the production plan for W2.
</Activity>

<Support title="Laser job going wrong?">

**Parts not cutting all the way through?** The most common causes: (1) power too low for this material thickness — increase power by 5–10%; (2) speed too high — reduce speed; (3) material is warped, lifting off the bed — use hold-down pins or tape the edges flat.

**Excessive charring on edges?** Speed is too low or power too high. Increase speed or reduce power. A slight char on birch ply is normal and sands off cleanly — heavy char means the settings need adjusting.

**File looks wrong in the laser software?** Check the scale: if the part outlines are the wrong size in the laser preview, the file units may have been misread. Export the file again with explicit mm units (in Inkscape: File → Document Properties → ensure 'mm' is set, then re-export SVG).
</Support>

Before your next session, identify the grain direction in your birch plywood. Plywood has alternating grain direction in each layer — how does this affect its strength in different directions? How does it affect which way the laser cuts most cleanly? Is this relevant to any joint or dimension in your product?
