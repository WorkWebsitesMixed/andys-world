<Section title="The three formulas — guaranteed recall" icon="🔢" id="formulas">

### Formula 1: Current-limiting resistor

R = (Vsupply - V_LED) / I_LED

Steps:
1. Convert I_LED from mA to A (divide by 1,000).
2. Subtract V_LED from Vsupply.
3. Divide the result by I_LED in A.
4. Choose the nearest standard value ABOVE the result (never below).

Standard E12 values (Ω): 100, 120, 150, 180, 220, 270, 330, 390, 470, 560, 680, 820, 1000, 1200, 1500, 1800, 2200, 2700, 3300, 3900, 4700.

### Formula 2: Voltage divider

Vout = Vsupply x R2 / (R1 + R2)

R1 = top resistor (between supply and output junction).
R2 = bottom resistor (between output junction and 0 V).

Steps:
1. Convert any kΩ values to Ω.
2. Calculate R1 + R2.
3. Calculate R2 / (R1 + R2).
4. Multiply by Vsupply.
5. Sense check: 0 V less than Vout less than Vsupply.

### Formula 3: Ohm's Law

V = I x R (voltage = current x resistance)

Rearrangements: I = V / R; R = V / I.

In Paper 4, Ohm's Law is most commonly used to verify the current through a resistor after choosing a standard value.

</Section>

<Section title="IPO labelling: no marks for guessing" icon="⚡" id="ipo">

### The three components and their roles

| Component | IPO role | Why |
|---|---|---|
| Switch | Input | The user (or sensor) controls the circuit through the switch — it is the trigger |
| Resistor | Process | It controls (processes) the current flowing to the output — limits LED current |
| LED, lamp, buzzer, motor | Output | It produces the visible/audible/physical effect — the result the user wants |

### When there is a sensor (thermistor, LDR)

The sensor is still the Input — it senses the environment and provides a signal to the process stage.

If a transistor is present: the transistor is the Process (it amplifies and switches the current). The resistor (if shown separately from the sensor divider) may still be Process stage.

### What examiners look for

- Correct labelling of all three sections: I, P, O (or Input, Process, Output).
- Correct assignment: switch/sensor = I; resistor/transistor = P; output device = O.
- If the question asks to "draw and label the IPO model" — draw a block diagram with labelled arrows, not a circuit diagram.

</Section>

<WorkedExample title="Five electronics calculations with full working">

**Q1 — Resistor for blue LED:**
Supply = 9 V. V_LED = 3.2 V. I_LED = 20 mA.
I_LED = 20 / 1000 = 0.020 A.
R = (9 - 3.2) / 0.020 = 5.8 / 0.020 = 290 Ω.
Nearest E12 above 290 Ω: 330 Ω.

**Q2 — Voltage divider (LDR in dark):**
Vsupply = 6 V. LDR (R1) = 80 kΩ = 80,000 Ω. Fixed resistor (R2) = 4 kΩ = 4,000 Ω.
Vout = 6 x 4,000 / (80,000 + 4,000) = 6 x 4,000 / 84,000 = 6 x 0.0476 = 0.286 V.
Transistor: 0.286 V is less than 0.6 V — transistor OFF.

**Q3 — Find current through 390 Ω resistor, 9 V supply, 2 V LED:**
Voltage across resistor = 9 - 2 = 7 V.
I = V / R = 7 / 390 = 0.01795 A = 17.95 mA.
Sense check: 17.95 mA is less than 20 mA — LED safe.

**Q4 — Find Vout for given sensor change:**
R1 (thermistor, cold) = 10,000 Ω. R2 (fixed) = 1,000 Ω. Vsupply = 9 V.
Vout = 9 x 1,000 / (10,000 + 1,000) = 9 x 1,000 / 11,000 = 0.818 V.
Transistor: 0.818 V > 0.6 V — transistor ON.

**Q5 — Find Vout for same sensor, hot:**
R1 (thermistor, hot) = 500 Ω. R2 = 1,000 Ω. Vsupply = 9 V.
Vout = 9 x 1,000 / (500 + 1,000) = 9 x 1,000 / 1,500 = 6 V.
Transistor: 6 V >> 0.6 V — transistor firmly ON.
</WorkedExample>

<Activity title="Electronics calculation drill" time="90 min" grouping="Individual" materials={["Calculator", "Electronics question bank (teacher-provided)", "Formula card (self-written from memory)"]}>

**Formula recall (0–10 min):**
1. Write all three electronics formulas from memory: resistor formula, voltage divider, Ohm's Law.
2. Write the E12 standard resistor values from 100 Ω to 4,700 Ω from memory.
3. Check both lists against the reference on this page. Correct any errors.

**Calculation drill (10–65 min):**
4. Complete ten electronics calculations from the question bank. For each:
   - Step 1: Convert units (mA to A; kΩ to Ω) before writing the formula.
   - Step 2: Write the formula.
   - Step 3: Substitute.
   - Step 4: Calculate.
   - Step 5: Sense check.
5. Self-mark after each pair of questions (not at the end of all ten).

**IPO labelling (65–80 min):**
6. For four given circuit diagrams: label each component with its IPO role. One sentence justification for each label.

**Error analysis (80–90 min):**
7. For any calculation that was wrong: identify the exact step where the error occurred. Was it unit conversion? Formula? Substitution? Arithmetic? Write a note for the inside cover of your folder.
</Activity>

<Support title="Electronics calculation support">

**Always convert units as step zero.** Before writing the formula, write: "I_LED = [value] mA = [value / 1000] A" and "R = [value] kΩ = [value x 1000] Ω". Write these on the answer sheet even before writing the formula. This makes the conversion visible to the examiner (earns method marks) and prevents substituting the unconverted value.

**Voltage divider: which is R1 and which is R2?** R1 is the top resistor — the one connected between the supply and the output junction. R2 is the bottom resistor — connected between the junction and 0 V (earth). In the formula, Vout = Vsupply x R2 / (R1 + R2) — the output voltage appears across R2.

**Sense check habit:** After every calculation, write one line: "Sense check: [result] is [within expected range / not within expected range]." If not within range, re-examine before submitting the answer.
</Support>
