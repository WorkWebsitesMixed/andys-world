Third open project session. By now, the pattern should be clear: identify the highest-impact task, work on it with full focus, record progress. Use that pattern again.

<Callout type="exam" title="Why this matters" id="why">
  <ExamLink to="project">Component 2</ExamLink> is worth **50% of your IGCSE**. Consistent use of catch-up sessions compounds over six sessions into substantial additional evidence.
</Callout>

<KeyTerms terms={[
  { term: "Diminishing returns", definition: "The principle that additional effort on an already-strong criterion earns fewer extra marks than the same effort applied to a weak criterion. When a criterion is already scoring well, time is better spent on the weakest criterion." },
]} />

<Section title="How to use this session" icon="🔧" id="session">

### Mid-catch-up folder review

At this point in the catch-up sequence, do a brief folder review:

| Criterion | Evidence present? | Estimated completeness |
|-----------|-----------------|----------------------|
| C3 | Ideas, matrix, justification | ___% |
| C4 | Model photo, tests, development | ___% |
| C5 | Working drawings, cutting list, plan | ___% |
| C6 | Making log W2–W8 + photos | ___% |
| C7 | Test results, strengths, weaknesses, improvements | ___% |

The criterion with the lowest estimated completeness is today's priority — unless it is close to 100%, in which case move to the next lowest.

</Section>

<Activity title="Open project session — individual" time="45 min" grouping="Individual" materials={["All project materials as needed", "Making log", "Camera", "Tools and materials as required"]}>

**Folder review (0–5 min):**
1. Fill in the table above for your own folder.
2. Identify the lowest-completeness criterion. Write today's priority.

**Work (5–40 min):**
3. Work on the priority task.

**Progress note (40–45 min):**
4. Write what was achieved and what remains.
</Activity>
