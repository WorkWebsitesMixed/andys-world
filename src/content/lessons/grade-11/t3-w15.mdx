<Section title="Section A triage in practice" icon="⚡" id="triage">

### The triage workflow

For every Section A question:
- **Under 10 seconds:** If you know the answer immediately, write it and move on.
- **10–30 seconds:** If you need to think but it's coming back to you, write and move on.
- **Over 30 seconds:** Write a one-word placeholder (the term or formula that feels relevant) and put a tick in the margin. Move to the next question.

Second pass: return only to ticked questions. You now have extra time (from the questions answered quickly) to think about the trickier ones.

### What to write during the second pass

If you still cannot recall the full answer:
- Write the formula or key term that is most relevant.
- Write a comparative statement if you cannot remember the absolute value.
- Write a partial explanation — any correct fact about the topic earns a method mark if the question is multi-step.

</Section>

<Activity title="40-mark Section A speed drill" time="45 min" grouping="Individual (exam conditions)" materials={["40-mark Section A drill paper (teacher-provided)", "Calculator", "Black pen"]}>

**Timed drill (0–35 min):**
1. Complete the 40-mark Section A paper in 35 minutes. Strict time limit — stop at 35 minutes regardless of completion.
2. Triage throughout: write and move on for fast questions; placeholder-and-tick for slow questions; second pass for ticked items in remaining time.

**Self-mark (35–42 min):**
3. Mark using the mark scheme provided. Score: _____ out of 40.

**Error categorisation (42–45 min):**
4. For every mark lost: label: (a) knowledge gap, (b) vocabulary/generic word, (c) calculation error (wrong arithmetic), (d) unit missing, (e) time (not reached in 35 minutes).
5. Write the count for each category. Which is highest?
</Activity>
