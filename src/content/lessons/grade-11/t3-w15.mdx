This is open project time. Every student's project is at a different stage — some need to complete the physical product, some need to write the evaluation, some need to photograph and document. Your job is to identify your most important unfinished task and make meaningful progress on it.

<Callout type="exam" title="Why this matters" id="why">
  <ExamLink to="project">Component 2</ExamLink> is worth **50% of your IGCSE**. Every bonus session you use for project completion is additional evidence for C6 or C7. Evidence that does not exist in the folder does not earn marks — additional making time is additional folder content.
</Callout>

<Section title="How to use this session" icon="🔧" id="session">

### Identify your priority task

Answer these three questions:

1. Which criterion in my folder is most incomplete right now — C3, C4, C5, C6, or C7?
2. What is the single task that would most improve that criterion's evidence?
3. Can I complete or significantly advance that task in 45 minutes?

Then work on that task for the full session.

### Common catch-up priorities

- **Making log incomplete:** Write missing log entries from memory. Add any available photographs. A complete log earns more than a partial log even if the photographs are reconstructed.
- **Product not finished:** Identify the one or two remaining steps and complete them. Prioritise: finish coat 2, circuit integration, final assembly.
- **Working drawings incomplete:** Add a missing view, a missing title block field, or missing dimensions.
- **Evaluation not written:** Draft the C7 evaluation in bullet points — convert to full sentences later.
- **No in-use photograph:** Set up the photography station and take one in-use shot.

</Section>

<Activity title="Open project session — individual" time="45 min" grouping="Individual (teacher circulates to check priority task is appropriate)" materials={["All project materials as needed", "Making log", "Camera", "Tools and materials as required by individual task"]}>

**Priority identification (0–5 min):**
1. Write at the top of your working page: "My priority task today is ___. When complete, it will improve my ___ [criterion] mark by ___."

**Work (5–40 min):**
2. Work on the priority task. No distractions. Full focus for 35 minutes.

**Progress note (40–45 min):**
3. Write a brief progress note in your making log: what was achieved, what remains.
</Activity>

<Support title="Not sure what to prioritise?">

Ask yourself: if the examiner saw my folder right now, which criterion would score lowest? Whatever the answer is — C5 (planning), C6 (making), C7 (evaluation) — that is where this session's 45 minutes should go.
</Support>
