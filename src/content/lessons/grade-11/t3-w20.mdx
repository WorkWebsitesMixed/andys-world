### Instructions

Cover this section. On a blank sheet, write every formula from memory. Then uncover and check. For any formula that was wrong or missing: write it five times from memory before reading any more of this page.

### Formulas (uncover to check)

**Resistor:** R = (Vsupply - V_LED) / I_LED [convert mA to A first; choose nearest E12 above]

**Voltage divider:** Vout = Vsupply x R2 / (R1 + R2) [R1 = top; R2 = bottom; 0 V less than Vout less than Vsupply]

**Ohm's Law:** V = I x R [I = V/R; R = V/I]

**Factor of Safety:** FoS = Ultimate load / Working load [FoS greater than 1.0 for safe design]

**Stress:** Stress = Force / Area [N and mm² → N/mm² = MPa]

**Beam reaction:** RA = W x (distance from load to B) / span; RA + RB = W

<Section title="The complete vocabulary card set — write from memory first" icon="✏️" id="vocab">

### Instructions

For each generic word: write the specific replacement from memory. Then check.

### Vocabulary replacements (uncover to check)

| Generic word | Replacement (specific property + value where applicable) |
|---|---|
| Strong | High tensile strength — mild steel approximately 400 MPa |
| Light | Low density — aluminium 2.7 g/cm³ (one-third of mild steel 7.8 g/cm³) |
| Rigid | High Young's modulus — steel 200 GPa, aluminium 70 GPa, MDF approximately 7 GPa |
| Durable | High hardness (for scratch resistance) OR high corrosion resistance (for outdoor/wet) — specify which |
| Easy to work | High machinability — MDF can be laser-cut without grain splitting |
| Safe | SELV (below 50 V DC) for electrical; chamfered/rounded edges for mechanical |
| Sustainable | Name the specific R — Reduce (nesting), Reuse (KD fittings), Recycle (single material), Repair (screws) |

<Activity title="Formula bank and vocabulary card production" time="45 min" grouping="Individual" materials={["A5 cards or small paper sheets (to make physical cards)", "Pencil for drafts, black pen for final cards"]}>

**Formula recall (0–15 min):**
1. Cover the formulas section above. Write all six formulas from memory on a blank sheet.
2. Uncover and check. Rewrite any incorrect formula five times from memory.
3. On a card (A5 or smaller): write the formula on the front, a worked example on the back. Make six formula cards.

**Vocabulary cards (15–30 min):**
4. Cover the vocabulary section. Write the replacement for each generic word from memory.
5. Check. For any wrong replacement: write it five times from memory.
6. Make seven vocabulary cards: generic word on front, specific replacement with value on back.

**Key values cards (30–40 min):**
7. Make a single card listing all eight key values. Write from memory first, then check.

**Interleaved retrieval test (40–45 min):**
8. Shuffle all cards. Draw three at a time from random positions in the deck. Answer: formula, vocabulary replacement, or value as appropriate. Check after each draw. Record any remaining errors for home study.
</Activity>
