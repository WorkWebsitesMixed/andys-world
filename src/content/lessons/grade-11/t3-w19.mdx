Penultimate catch-up session. After this, there is one more (B10). Use your B8 progress note to decide what must happen today vs. what can happen in B10.

<Callout type="exam" title="Why this matters" id="why">
  <ExamLink to="project">Component 2</ExamLink> is worth **50% of your IGCSE**. This is the second-to-last unstructured project session. The folder closes after B10.
</Callout>

<Section title="How to use this session" icon="🔧" id="session">

### Two-session planning

With B9 and B10 remaining, divide what is left:

1. **B9 (today):** the task that takes the most time or requires the most concentration
2. **B10 (next):** the task that can be done in 45 minutes as a final check and tidy

### Final folder order check

By the end of B9, your folder should contain every page in the correct order:
- C1: Research
- C2: Design brief and specification
- C3: Idea generation, development, matrix, justification
- C4: Model photographs, test results, design changes, sustainability
- C5: Working drawings, cutting list, production plan
- C6: Making log (T3 W2–W8) with photographs
- C7: Test results, strengths, weaknesses, proposed improvements

If any section is missing a page, today is the day to add it.

</Section>

<Activity title="Open project session — individual" time="45 min" grouping="Individual" materials={["All project materials as needed", "Making log", "Camera", "Folder"]}>

**Two-session plan (0–5 min):**
1. Write: "B9 task: ___. B10 task: ___."
2. Start B9 task immediately.

**Work (5–40 min):**
3. Full focus on B9 task.

**Progress note (40–45 min):**
4. Write what was achieved. Confirm the B10 task is realistic.
