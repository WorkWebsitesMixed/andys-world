A dedicated Section B practice session with an unseen design context. Section B is where students with strong knowledge often fail to convert that knowledge into marks — because they write about the topic in general rather than about THIS context in particular. This session trains context-specific application.

<KeyTerms terms={[
  { term: "Unseen context", definition: "A design scenario in Section B that students have not encountered before — they must apply their knowledge to a new situation rather than recalling a prepared example. Success in unseen contexts requires understanding principles (why things work) rather than memorising examples (what things are). The principle applies universally; the memorised example does not." },
  { term: "Context anchoring", definition: "The practice of beginning every Section B answer with a reference to the specific design context: the material named, the user described, the product function stated. Context anchoring ensures the answer is about THIS product, not a generic product. It typically earns 1 mark per mention — and prevents the 'generic answer' mark loss." },
  { term: "Extrapolation", definition: "Applying a principle known from one context to a new context. In Section B, a student who understands that triangulation resists racking forces can extrapolate this to an unseen flat-pack furniture context without having studied flat-pack furniture specifically. Extrapolation is the core exam skill of Section B." },
]} />

<Section title="Approaching an unseen Section B context" icon="📖" id="approach">

### Step 1: Read the full context (3 minutes)

Before reading the questions, read the full Section B scenario:
- What is the product?
- Who is the user?
- What are the function requirements?
- What materials or processes are mentioned?
- Are there any dimensions or loads given?

Write a 5-word summary in the margin: "wall shelf, 3 kg, student."

### Step 2: Read all sub-parts (2 minutes)

Read every sub-part before answering any. Note: which sub-part asks about structure? Which about materials? Which requires a calculation? Plan the sequence: calculations first (fixed method, not dependent on previous answers) or explanation questions first (use established context to anchor answers).

### Step 3: Context-anchor every sentence

The first sentence of every Section B answer should contain: the product name, the material or process being discussed, and the specific function or user requirement it relates to. Example: "The steel bracket in the wall-mounted shelf must resist the bending moment caused by the 3 kg static load at the shelf midspan."

### Step 4: Margin plan for 4+ mark questions

6-mark question: 6 bullet points in the margin before writing a single sentence. Check: are all 6 different? Check: does each one relate to THIS specific context?

</Section>

<Activity title="Full Section B unseen context practice" time="45 min" grouping="Individual (exam conditions)" materials={["Section B unseen context paper (teacher-provided)", "Calculator", "Black pen"]}>

**Read and plan (0–5 min):**
1. Read the full context. Write your 5-word summary in the margin.
2. Read all sub-parts. Number them. Note mark values.

**Calculations (5–15 min):**
3. Complete any calculation sub-parts first (fixed method, no dependency on other sub-parts).

**Explanation questions (15–40 min):**
4. For each explanation question: context-anchor the first sentence. Margin plan for 4+ mark questions. Write the answer. Stop when all mark points are covered.

**Check-back (40–45 min):**
5. Sub-parts all answered? Units on calculations? Context anchor in first sentence of each answer?

**Self-mark (after session):**
6. Score using mark scheme. For each missed mark: is it a knowledge gap or a context-specificity failure?
</Activity>
