A working drawing communicates manufacturing instructions to anyone who needs to make your product — not just to you. This session teaches first-angle orthographic projection: the British standard for engineering drawings that IGCSE D&T Paper 4 tests directly.

<KeyTerms terms={[
  { term: "Orthographic projection", definition: "A drawing method in which an object is represented by two or more views, each showing the object as seen from a specific direction (front, side, plan/top). All views are aligned and at the same scale. Lines in one view project directly into corresponding positions in adjacent views." },
  { term: "First-angle projection", definition: "The UK standard for orthographic drawing. The side view appears on the LEFT when looking from the RIGHT. The plan view appears BELOW the front view. Identified by the first-angle projection symbol. Used in IGCSE D&T." },
  { term: "Hidden detail line", definition: "A dashed line (short dashes, evenly spaced) used to show edges or outlines that exist behind the visible face of the view. Hidden detail lines follow the same projection rules as visible lines and must be accurately projected from other views." },
  { term: "Centre line", definition: "A thin dash-dot line passing through the centre of circular features (holes, cylinders, arcs). Centre lines appear in all views where the circular feature is projected. They extend slightly beyond the outline of the feature." },
  { term: "Title block", definition: "A standard information block added to every working drawing. Contains: drawing title, scale, date, designer's name, material, finish, and a projection symbol. Positioned in the lower-right corner of the drawing sheet." },
]} />

<Section title="Setting up a first-angle orthographic drawing" icon="📐" id="setup">

### The three views

An orthographic drawing shows at minimum three views:
1. **Front view (elevation):** The most informative view — shows the main face of the object as seen from the front.
2. **Side view (end elevation):** Positioned to the LEFT of the front view in first-angle projection.
3. **Plan view (top view):** Positioned BELOW the front view (first-angle: the view "falls away" from the observer).

### Projection lines

Horizontal and vertical projection lines connect corresponding points between views. Every corner of the front view projects horizontally into the side view and vertically into the plan view. These projection lines are drawn as construction lines (thin, light).

### The 45-degree miter line

A 45° construction line is drawn at the corner between the side view position and the plan view position. This transfers the depth dimension from the side view into the plan view without measuring it twice.

### Line types

| Line type | Appearance | Use |
|---|---|---|
| Visible outline | Continuous thick (0.5–0.7 mm) | Visible edges of the object |
| Projection/construction | Continuous thin (0.25 mm) | Layout lines — project between views |
| Hidden detail | Dashed thin | Edges not visible in this view |
| Centre line | Thin dash-dot | Centre of holes, symmetry axes |
| Dimension line | Thin, with arrowheads | Shows measured dimension |

</Section>

<Section title="Dimensioning" icon="📏" id="dimensioning">

### Rules for dimensioning

1. **Dimension lines** are thin lines with arrowheads at each end, placed outside the outline of the view.
2. **Extension lines** project from the object outline to the dimension line — leave a 2 mm gap.
3. **Dimension values** are placed above the dimension line, centred between the arrowheads. All values in mm.
4. **Do not duplicate** dimensions — each measurement appears on only one view.
5. Use **baseline dimensioning** (all from one datum point) rather than chain dimensioning wherever possible.

### What to dimension

For the Component 2 main structural component, the minimum three dimensions:
- Overall length (longest axis)
- Overall width
- Overall height or thickness

Additional for manufacture: joint dimensions (depth of dado, width of rebate), hole diameters, corner radii.

</Section>

<WorkedExample title="Three-view first-angle orthographic of a shelf bracket">

**Object:** L-shaped shelf bracket. 200 mm wide horizontal arm, 150 mm tall vertical arm, 18 mm thick throughout.

**Front view:** Shows the L-shape — horizontal arm width (200 mm), vertical arm height (150 mm). Visible outlines thick continuous.

**Side view (to left of front view):** A rectangle 18 mm wide x 168 mm tall showing the bracket thickness.

**Plan view (below front view):** Shows the bracket from above — a rectangle 200 mm wide x 18 mm deep for the horizontal arm plus 18 mm x 18 mm for the vertical arm top.

**Projection:** Horizontal lines from front view project into side view (same vertical positions). Vertical lines from front view drop into plan view (same horizontal positions). 45° miter transfers depths.

**Dimensions:** 200 mm horizontal (front view), 150 mm vertical (front view), 18 mm thickness (side view).

**Title block:** Title: "Shelf bracket — Component 2". Scale: 1:5. First-angle symbol. Material: 18 mm MDF.
</WorkedExample>

<Activity title="First-angle orthographic drawing of main component" time="90 min" grouping="Individual" materials={["A3 paper", "Ruler, 30-60-90 set square, 45-degree set square", "2H pencil for construction, HB for outlines"]}>

**Setup (0–15 min):**
1. Choose the main structural component of your Component 2 product.
2. Choose a scale that fills the A3 sheet comfortably with all three views and a title block (typically 1:2 or 1:5).
3. Plan the layout: mark positions of front view, side view (left of front), plan view (below front), title block (lower right).

**Front view (15–35 min):**
4. Draw the front view as thin construction outline first. Check proportions against scale.

**Side and plan views (35–60 min):**
5. Use projection lines to position all key points in the side and plan views.
6. Draw the 45° miter line and use it to transfer depth from the side view into the plan view.

**Line weights and detail (60–75 min):**
7. Apply final line weights: thick continuous for visible outlines, thin dashed for hidden detail, thin dash-dot for centre lines.
8. Add three or more dimensions with correct dimension lines, extension lines, and arrowheads.

**Title block (75–85 min):**
9. Rule the title block. Fill in: title, scale, projection symbol, material, name, date.

**Self-check (85–90 min):**
10. Side view to the LEFT of front view? Plan view BELOW front view? Projection lines correctly aligned? Title block complete?
</Activity>

<Support title="Common orthographic errors">

**Side view ends up on the right:** In first-angle projection, the right-side view goes to the LEFT (the view falls through the object to the opposite side). Double-check using the first-angle projection symbol.

**Plan view above the front view:** In first-angle, the view from above is placed BELOW the front view. First-angle = views fall away from the observer.

**Dimensions duplicated in two views:** State each dimension once only — in the view where it is most clearly seen.

**Missing hidden detail:** Ask: are there any edges inside the object that cannot be seen in this view direction? If so, they must appear as dashed lines.
</Support>
